從零開始使用 Github Page建立 Hexo個人部落格

1.install node.js
2.install git
3.create new folder folder
4.enter folder
5.right click and select Git Bash Here
6.enter command line: $ npm install -g hexo-cli
7.enter command line: $ hexo init
//8.enter command line: $ npm install
9.enter command line: $ hexo server

go to local page: http://localhost:4000/

10.enter command line: $ hexo generate (generate at folder “public”)
11.modify deploy setting at _config.yml
deploy:
type: git
repository: git@github.com:Mortiz09/mortiz09.github.io.git
branch: master
12.enter command line: $ npm install hexo-deployer-git –save

13-1.enter command line: $ git config –global user.name “mortiz09“
13-2.enter command line: $ git config –global

user.email “mortiz09@hotmail.com“
13-3.enter command line: $ ssh-keygen -t rsa -C “q90036@hotmail.com”
13-4.open C:\Users\Moz.ssh\id_rsa.pub, copy all text
13-5.go to github, Settings, SSH and GPG keys
13-6.click “New SSH key”, paste to input , click “Add SSH key”
(ref: http://blog.csdn.net/jingtingfengguo/article/details/51892864)

14.enter command line: $ hexo deploy

15.go to your page: https://mortiz09.github.io/
