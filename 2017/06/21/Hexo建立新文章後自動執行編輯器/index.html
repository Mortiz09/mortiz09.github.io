Hexo建立新文章後自動執行編輯器

2017-06-21

每次執行完 $ hexo new後都要去 \source\_posts裡面找出 .md檔，再用編輯器開啟真的是超麻煩的
這些功夫當然是能省則省

  1. 在 Hexo folder內建立新資料夾 scripts
  2. 在 scripts資料夾底下新增一個 .js檔案，檔案內容如下
    var exec = require(‘child_process’).exec;
    hexo.on(‘new’, function(data) {
    exec(‘start “C:\Users\XXX\AppData\Roaming\Haroo Studio\Haroopad\haroopad.exe“ ‘+ data.path);
    });
  3. 在 Git Bash執行 $ hexo new指令看看

最後找出錯誤是因為我在程式碼最後 }); 又隔一行然後放了註解 …
